# 🍔 Real-World-Swiggy Data Analysis & Excel Dashboard

This project analyzes **Swiggy food delivery sales data** using Python and presents insights through an **interactive Excel dashboard**. The goal of this project is to demonstrate **data analysis, visualization, and dashboard creation skills** using real-world style data.

---

# 📊 Project Overview

The project focuses on analyzing food delivery sales data to understand:

- Sales performance
- Customer ratings
- Order patterns
- Food category popularity
- Sales by city and state
- Time-based sales trends

The analysis was performed using **Python libraries**, and the results were visualized using an **Excel Dashboard**.

# 📈 Excel Dashboard Features

The dashboard provides an interactive overview of Swiggy sales performance.

### Key Metrics
- **Total Sales:** ₹53.01M
- **Average Rating:** 4.34
- **Average Order Value:** ₹268.51
- **Rating Count:** 5.59M
- **Total Orders:** 197.43K

### Visualizations Included
- Monthly Sales Trends
- Daily Sales Trends
- Weekly Sales Trends
- Sales by Food Type
- Sales by State
- Top 5 Cities by Sales
- Quarterly Sales, Ratings & Orders

### Dashboard Filters
Users can filter the dashboard using:

- Month
- Category
- Restaurant Name

# 📊 Key Insights

Some insights derived from the data analysis:

- Sales show **steady growth across months**.
- Major contribution comes from **top metropolitan cities**.
- **Food categories like Chinese, Pasta, and Rolls** generate significant sales.
- Average customer ratings remain **above 4**, indicating strong customer satisfaction.
